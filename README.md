# WAR
*This is a game app that was built during my time at Nashville Software School as a class excercise. This is a simple card game of War*
<br>
### Rules
The deck is divided evenly among the players. The player with the higher card takes both of the cards played and moves them to their stack. If two cards are equal than it's a draw.

### Tools
- [Deck of Cards](http://deckofcardsapi.com/) API
- RequireJS
- jQuery
- Firebase
