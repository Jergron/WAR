# WAR
*This is a game app that was built during my time at Nashville Software School as a class exercise. This is a simple card game of War*
<br>
### Rules
A deck of 52 cards is given to each player. The player with the higher card gets 2 points. If two cards are equal than it's a draw. The player with the most points wins the game. The game ends when both players run out of cards. 

### Tools
- [Deck of Cards](http://deckofcardsapi.com/) API
- RequireJS
- jQuery
- Firebase
